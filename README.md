## Tunneling with Localtunnel

To make your local Next.js server accessible over the internet, you can use Localtunnel. Follow these steps:

### Run the Next.js server:

```bash
pnpm run dev
```

```bash
pnpm i -g localtunnel
```

```bash
lt -s podie --port 3000
```
